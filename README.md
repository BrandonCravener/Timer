# [Timer](https://t.brandoncravener.com)

## Description

A simplistic timer web app that was created as an alternative to e.ggtimer.com.

---

## Features

- Simple UI
- Accurate
- Animations
- URL Parsing

---

## Installation

Install this project by running:

1. `$ git clone https://github.com/BrandonCravener/Timer`
2. `$ cd Timer`
3. `$ yarn`

Run with `$ yarn start`
